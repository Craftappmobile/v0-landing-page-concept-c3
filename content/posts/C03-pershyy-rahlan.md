# МК: Перший реглан — як зв'язати светр без помилок

## Вступ

Майстер-клас для початківців: перший светр регланом зверху. Покроково з калькулятором, без складних термінів.

## Чому реглан — найкращий вибір для першого светра

Без швів, можна приміряти в процесі, легко виправляти помилки. Не потрібно зшивати деталі.

## Що потрібно знати перед початком

Базові навички: набір петель, лицьова/виворітна петля, накид. Все інше — у процесі.

## Покроковий план

1. Виміряйте обхват шиї. 2. Зв'яжіть зразок. 3. Використайте калькулятор реглану. 4. Наберіть петлі. 5. Зв'яжіть росток. 6. Виконуйте добавки по регланних лініях. 7. Відділіть рукави. 8. Завершіть тіло і рукави.

## Найчастіші помилки новачків

Забули росток (светр «душить» спереду). Неправильно розподілили петлі (рукави завузькі). Не маркували регланні лінії.

## Типові помилки

1. **Пропуск зразка.** Навіть у майстер-класах важливо перевіряти свою щільність.
2. **Неправильний вибір пряжі.** Для перших проєктів обирайте світлу, гладку пряжу — на ній краще видно петлі.
3. **Відсутність маркерів.** Використовуйте маркери для позначення важливих точок.
